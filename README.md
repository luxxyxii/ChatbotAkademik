# Chatbot Akademik Berbasis Machine Learning

Chatbot berbasis website yang membantu mahasiswa dalam mengelola tugas, deadline, dan motivasi akademik menggunakan pendekatan Machine Learning (Intent Classification).

## Fitur
- Intent classification berbasis TF-IDF & Naive Bayes
- Bahasa Indonesia
- Tampilan chat mirip WhatsApp
- Berbasis website

## Teknologi
- Python
- Flask
- Scikit-learn
- HTML, CSS, JavaScript
